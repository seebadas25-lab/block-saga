# Contributing to Block Saga

Thank you for your interest in contributing to Block Saga!

## How to Contribute

1.  **Fork the repository** on GitHub.
2.  **Clone** your fork locally.
3.  **Create a branch** for your feature or fix.
4.  **Commit** your changes.
5.  **Push** to your fork.
6.  Submit a **Pull Request**.

## Development Setup

1.  Move to the project directory: `cd block-saga`
2.  Install dependencies: `npm install`
3.  Start server: `npm run dev`

## Code Style

-   Use **Vanilla JS** (Module syntax).
-   Keep the Canvas rendering optimized (avoid heavy allocations in game loop).
-   Follow the existing specific folder structure in `src/game/`.
